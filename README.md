# 🎬 Media Hub - Social Creator Platform

A web-based media platform showcasing reels and content from Black American and African content creators across TikTok, Instagram, and YouTube.

## 📋 Quick Start (Beginner-Friendly Steps)

### **STEP 1: Get the Project on Your Computer**

1. **Click the green "Code" button** on GitHub (top of this page)
2. **Click "Download ZIP"**
3. **Find the file on your computer** (probably in Downloads folder)
4. **Right-click → Extract All** (or just double-click)
5. **You now have the project!** ✅

### **STEP 2: Open the Project in Your Text Editor**

1. **Download free editor:** Go to https://code.visualstudio.com/
2. **Install** it (follow the simple steps)
3. **Open the editor**
4. **Click File → Open Folder**
5. **Find and open your Media-hub folder**
6. **You're in!** ✅

### **STEP 3: Set Up Your Project Files**

1. **Right-click in the empty space** on the left side of your editor
2. **Create new file → type `index.html`**
3. **Create new file → type `style.css`**
4. **Create new file → type `script.js`**
5. **You now have 3 files!** ✅

## 🚀 What This Project Does

### **Features:**
- ✅ Shows videos from TikTok creators
- ✅ Shows videos from Instagram creators  
- ✅ Shows videos from YouTube creators
- ✅ Easy search by creator name
- ✅ No manual uploading needed (automatic!)
- ✅ Mobile-friendly (works on phones)
- ✅ Potential to add ads for money
